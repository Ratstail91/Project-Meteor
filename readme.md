## Blue Harvest

Blue Harvest is an open source SDL/OpenGL dungeon crawler video game. All of the information about this game can be found in "Documentation/Game Design Document.doc”

## Instructions

This game is built using SDL and OpenGL. To build the solution, open the project using Visual Studio, switch the configuration to “Release”, and build the solution.
